Как удалить программное обеспечение Picasa 3?

Для удаления Picasa можно использовать компонент панели управления Windows "Установка и удаление программ" или команду удаления из группы программ Picasa ("Пуск" > "Программы" > "Picasa 3" > "Удалить").

В процессе удаления будет отображен запрос на удаление также базы данных Picasa. Выберите "Да", если хотите удалить структуру и изменения, сделанные в Picasa (если Вы не собираетесь повторно устанавливать Picasa 3). Нажмите кнопку "Нет", если необходимо сохранить сделанные Вами изменения (если Вы собираетесь переустановить Picasa 3).

После удаления на жестком диске могут оставаться некоторые файлы или папки из установки Picasa. Для удаления папки Picasa выполните следующие действия.

1. Откройте "Мой компьютер" > диск C: > "Program Files".
2. Нажмите правой кнопкой мыши папку "Picasa".
3. Выберите команду "Удалить".
4. Нажмите кнопку "Да".

Теперь программное обеспечение Picasa полностью удалено с компьютера.
