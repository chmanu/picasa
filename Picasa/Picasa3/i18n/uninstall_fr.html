Comment puis-je désinstaller Picasa 3 ?

Vous pouvez désinstaller Picasa à partir du Panneau de configuration Windows (option Ajout/Suppression de programmes) ou à l'aide de l'option de désinstallation dans le groupe de programmes Picasa (« Démarrer » > « Programmes » > « Picasa 3 » > « Désinstaller »).

Pendant la procédure de désinstallation, un message vous demande si vous souhaitez supprimer également la base de données de Picasa. Cliquez sur « Oui » si vous voulez effacer l'organisation de vos albums de photos ou les modifications que vous avez effectuées dans Picasa (dans le cas où vous ne souhaitez pas réinstaller Picasa 3), ou sur « Non » si vous voulez conserver toutes ces informations (dans le cas où vous envisagez de réinstaller Picasa 3).

Malgré la désinstallation, quelques fichiers ou dossiers créés lors de l'installation de Picasa peuvent demeurer sur le disque dur. Vous pouvez supprimer ces fichiers en procédant comme suit :

1. Double-cliquez sur « Poste de travail » et sélectionnez C:> « Program Files ».
2. Cliquez sur le dossier Picasa avec le bouton droit.
3. Cliquez sur « Supprimer ».
4. Cliquez sur « Oui ».

Picasa est ensuite totalement supprimé de votre ordinateur.
