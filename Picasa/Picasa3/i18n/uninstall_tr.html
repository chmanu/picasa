Picasa 3'yi bilgisayarımdan nasıl kaldırırım?

Picasa'yı Windows'un Kontrol Panelindeki Program Ekle/Kaldır aracı ile, veya Picasa Program grubundaki ("Başlat" > "Programlar" > "Picasa 3" > "Kaldır") Kaldır seçeneği ile kaldırabilirsiniz.

Picasa bilgisayarınızdan kaldırılırken, size Picasa veritabanını silmek isteyip istemediğiniz de sorulacak. Eğer Picasa'da yaptığınız düzenlemeleri veya değişiklikleri silmek istiyorsanız "Evet"i seçin (eğer Picasa 3'yi tekrar kurmayı düşünmüyorsanız). Yaptığınız düzenlemeleri silmek istemiyorsanız, "Hayır"ı tıklayın (eğer Picasa 3'yi tekrar kurmak istiyorsanız).

Program kaldırıltıktan sonra, Picasa programından geride kalan dosyalar veya klasörler olabilir. Picasa klasörünü tamamen silmek için:

1. "Bilgisayarım" > C: sürücüs > 'Program Dosyaları'nı açın.
2. Picasa klasörünü sağ tıklayın.
3. "Sil"i tıklayın.
4. "Evet"i tıklayın.

Picasa artık bilgisayarınızdan tamamen kaldırılmıştır.
