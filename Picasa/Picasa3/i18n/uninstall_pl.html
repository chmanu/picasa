Jak usunąć program Picasa 3?

Program Picasa można odinstalować albo z panelu sterowania Windows, korzystając z opcji „Dodaj / Usuń Programy” („Add / Remove Programs”), albo korzystając z opcji „Odinstaluj” („Uninstall”) w grupie programów Picasa (wybierając „Start”, a potem „Programy”, „Picasa 3” i „Uninstall”).

Podczas deinstalacji programu Picasa 3 zostaniesz poproszony o usunięcie jego bazy danych. Wybierz „Tak”, jeżeli chcesz usunąć wszelkie sposoby organizacji plików i inne zmiany, jakich dokonałeś korzystając z programu Picasa (przy założeniu, że nie masz zamiaru ponownie instalować tego programu). Kliknij „Nie”, jeżeli chcesz wszystkie te zmiany zachować (przy założeniu, że zamierzasz kiedyś ponownie zainstalować program Picasa 3).

Po usunięciu programu Picasa wciąż mogą pozostać pewne pliki lub foldery tego programu. Można je usunąć w następujący sposób:

1. Otwórz „Mój komputer” („My computer”), a następnie folder „Program Files” na dysku C:.
2. Kliknij prawym przyciskiem myszy w folder „Picasa”.
3. Kliknij „Usuń”.
4. Kliknij „Tak”.

Program Picasa zostanie całkowicie usunięty z Twojego komputera.
