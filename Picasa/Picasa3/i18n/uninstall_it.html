Come faccio a disinstallare Picasa 3?

Puoi disinstallare Picasa da Installazione applicazioni nel Pannello di controllo di Windows o utilizzando l'opzione Disinstalla nel gruppo di programmi Picasa ("Start" > "Programmi" > "Picasa 3" > "Disinstalla").

Durante la procedura di disinstallazione ti verrà chiesto di rimuovere anche il database di Picasa. Seleziona "Sì" se desideri eliminare tutte le organizzazioni dei file e le modifiche che hai apportato in Picasa (se non hai intenzione di reinstallare Picasa 3). Seleziona "No" se desideri conservare tutte le organizzazioni dei file e le modifiche che hai apportato (se hai intenzione di reinstallare Picasa 3).

Dopo la disinstallazione potrebbero essere ancora presenti alcuni file o cartelle dell'installazione di Picasa. Per rimuovere la cartella Picasa, procedi come indicato di seguito:

1. Apri "Risorse del computer" > Unità C: > "Programmi".
2. Fai clic destro sulla cartella Picasa.
3. Fai clic su "Elimina".
4. Fai clic su "Sì".

Picasa verrà rimosso completamente dal computer.
